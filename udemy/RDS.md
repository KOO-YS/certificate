# AWS RDS

- SQL을 쿼리 언어로 사용하는 관계형 데이터 베이스 서비스
- AWS에서 관리되는 클라우드 데이터베이스
    - Postgres
    - MySQL
    - MariaDB
    - Oracle
    - Microsoft SQL Server
    - Aurora (AWS proprietary database : AWS가 소유권을 가진 DB)
- **EC2 인스턴스 상에 자체 데이터베이스를 배포하지 않고 RDS를 사용하는 이유는?**
    - 데이터베이스 뿐만 아니라 여러 기타 서비스도 제공하기 때문
