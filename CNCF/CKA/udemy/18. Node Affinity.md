# Node Affinity
특정 Node에 Pod가 위치할 수 있도록 하는 개선된 기능 Node Affinity

### 같은 동작을 하는 `Node Selector`와 `Node Affinity` 정의 파일
- 요구사항 : Large Node에 Pod를 위치시킬 것

#### Node Selector 정의 파일
```yaml
apiVersion: v1
kind: Pod
metadata:
    name:
spec:
    containers:
    - name: my-container
      image: nginx
      nodeSelector:
          size: Large 
```

#### Node Affinity 정의 파일
```yaml
apiVersion: v1
kind: Pod
metadata:
    name:
spec:
    containers:
    - name: my-container
      image: nginx
    affinity:
      nodeAffinity:
        requiredDuringSchedulingIgnoredDuringExecution:
          nodeSelectorTerms:
          - matchExpressions:
            - key: size
              operator: In
              values:
              - Large
```
